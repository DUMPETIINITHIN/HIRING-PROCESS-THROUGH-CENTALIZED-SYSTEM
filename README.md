# 📌 HIRING PROCESS THROUGH CENTRALIZED SYSTEM

A centralized web-based application that streamlines the hiring cycle including applicant management, interviews, selection, and rejection—making recruitment easy, fast, and efficient for organizations.

---

## 🚀 Features

### 🔐 Authentication
- User Signup
- Login
- Role-based access (Admin/HR)

### 👥 Applicant Management
- Add Applicants
- View Applicants
- Update Profiles
- Track Status

### 🗓 Interview Module
- Schedule Interviews
- View / Update Results
- Manage interview details

### 🏆 Selection
- Approve selected candidates
- Move them to selection list
- Generate selection reports

### ❌ Rejection
- Mark rejected applicants
- Store rejection reason
- Review rejected profiles later

## 🧑‍💻 How to Run the Project

### Clone repository
```sh
git clone https://github.com/YOURUSERNAME/HIRING-PROCESS-THROUGH-CENTRALIZED-SYSTEM.git
```

### Enter project folder
```sh
cd HIRING-PROCESS-THROUGH-CENTRALIZED-SYSTEM
```

### Install requirements
```sh
pip install -r requirements.txt
```

### Run migrations
```sh
python manage.py migrate
```

### Start development server
```sh
python manage.py runserver
```
